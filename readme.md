gap — это CSS-свойство, которое задаёт отступы между дочерними элементами внутри контейнера.

| Критерий | gap | margin |
| :--- | :--- | :--- |
| Где задаётся | На контейнере | На каждом дочернем элементе |
| Внешние отступы | Не создаёт отступов снаружи контейнера | Создаёт (первый/последний элемент "выпирают") |
| Схлопывание | Не схлопывается | В блочной модели вертикальные margin схлопываются |
| Предсказуемость в flex/grid | Идеальная: движок сам распределяет пространство | Может ломать выравнивание, требует хаков (:not(:last-child)) |
| Производительность | Оптимизирован на уровне layout-движка | Требует пересчёта box-model для каждого элемента |
| Семантика | "Расстояние между элементами в группе" | "Отступ элемента от окружения" |
