Новые свойства:
1) gap — это CSS-свойство, которое задаёт отступы между дочерними элементами внутри контейнера.

2) ::befo / atter (псевдоэлементы) - Создаёт виртуальный элемент до или после содержимого блока, не трогая HTML. Держит разметку чистой, добавляет декор или функционал только через CSS.
Когда применять:
    - Кастомные маркеры списков
    - Иконки, бейджи, оверлеи, разделители
    - Эффекты при :hover (подчёркивание, подсветка, сдвиг)


3) transform - визуально трансформирует элемент: сдвигает (translate), вращает (rotate), масштабирует (scale), наклоняет (skew).

4) width: fit-content; - Сжимает блочный элемент до ширины его содержимого (как inline-block, но остаётся блоком). Решает проблему "блочный элемент всегда занимает 100% ширины". Позволяет центрировать блок через margin: 0 auto.
Когда применять:
    - Кнопки, карточки, теги, которые должны быть не шире текста
    - Центрирование блочных элементов без flexbox
    - Адаптивные инпуты/селекты

5) text-align: center; - Выравнивает по горизонтали строчное содержимое (текст, картинки, inline/inline-block элементы) внутри блока. Самый простой и нативный способ центрировать текст или кнопки-ссылки без лишних обёрток.
Когда применять:
    - Заголовки, абзацы, навигация, иконки в строку
    - Простые лендинги, где не нужен flex/grid
    - Когда нужно быстро выровнять текст внутри контейнера

6) :is() - это современная CSS-функция для группировки селекторов. Она позволяет перечислить несколько элементов через запятую внутри одних скобок, не дублируя родительскую часть селектора.
Пример:
    - без :is()
        .card h3,
        .card p,
        .card span,
        .card .link {
        color: #333;
        }
        
    - c :is()
        .card :is(h3, p, span, .link) {
            color: #333;
        }
